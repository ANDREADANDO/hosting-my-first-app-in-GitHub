# Calculator app

## Overview
This is a simple calculator app built using **Python** and **Streamlit**. It lets users perform basic arithmetic operations such as addition, subtraction, multiplication, and division through an easy-to-use web interface.

## Features
- Addition of two numbers  
- Subtraction of two numbers  
- Multiplication of two numbers  
- Division of two numbers (with error handling for division by zero)  

## How to Use the App Online
1. Open your web browser on **any device** (phone, tablet, or computer).  
2. Go to the live app link: [Streamlit Calculator](https://hosting-my-first-app-in-app-vewszzsaziwmkpseriybsb.streamlit.app/)  
3. Enter the first number in the input box.  
4. Enter the second number in the input box.  
5. Choose the operation from the dropdown menu: Add, Subtract, Multiply, or Divide.  
6. Press the **Calculate** button.  
7. The result will appear instantly below the button.
